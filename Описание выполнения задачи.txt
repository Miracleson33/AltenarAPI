Разработка сервиса (API) проводилась в MS Visual Studio Community 2019 на C#.

Для создания использована платформа ASP.NET Core 5.

В качестве СУБД использован локальный MS SQL Server Visual Studio.
Работа с базой данных организована через Entity Framework Core с использованием подхода code-first.

Структура базы данных состоит их 2-х таблиц:

1. IncomingMessages - Таблица для хранения сообщений получаемых в запросах от клиентов.
Статистика браузера (сессия, IP адрес, версии системы и браузера) берется из контекста запроса клиента.

2. Parameters - Таблица для хранения статистических параметров содержащихся в получаемых сообщениях.
Отношение таблиц  IncomingMessages и Parameters - один ко многим.

Для тестирования сервиса (API) и отправки POST запросов с сообщениями использовался Postman и интерфейс предоставляемый фреймворком Swagger UI.

Исходный код сервиса доступен на GitHub. AltenarAPI
